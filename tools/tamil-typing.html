Tamil Typing Tool

Type in Tamil using English keyboard - automatically converts English letters to Tamil

How to use:

  1. Type English letters in the box below
  2. They will automatically convert to Tamil as you type
  3. Example: Type 'vanakkam' → வணக்கம்

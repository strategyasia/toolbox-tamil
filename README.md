# ToolBox Tamil - உபகரணப் பெட்டி தமிழ்

**Free online tools for everyone with special features for the Tamil community**

🌐 **Live Demo:** [Coming Soon]

## 🎯 Features

### Universal Tools
- **PDF Tools**: Convert, merge, split, compress PDFs
- **Image Tools**: Compress, resize, convert, crop images
- **Text Tools**: Text-to-speech, word counter, case converter
- **Utility Tools**: QR code generator, barcode generator, color picker, password generator

### Tamil-Specific Tools 🇮🇳
- **Tamil Typing Tool**: Type Tamil using English keyboard
- **Tamil Unicode Converter**: Convert between Tamil fonts and Unicode
- **Tamil Transliteration**: Romanize Tamil text
- **Tamil Word Counter**: Count words and characters in Tamil
- **Tamil Calendar Converter**: Convert between Tamil and Gregorian calendars
- **Tamil Numeral Converter**: Convert numbers to Tamil numerals

## 🚀 Quick Start

### Local Development

1. Clone or download this repository
2. Open `index.html` in your browser
3. Or use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Python 2
python -m SimpleHTTPServer 8000

# Using Node.js (install http-server first: npm install -g http-server)
http-server -p 8000
```

4. Visit `http://localhost:8000`

### Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Push this code to the repository
3. Go to Settings → Pages
4. Select `main` branch and `/` (root) folder
5. Your site will be live at `https://yourusername.github.io/repository-name`

📊 Google Analytics

Track your website traffic and user behavior. Get your tracking ID from Google Analytics.

Format: G-XXXXXXXXXX (GA4) or UA-XXXXXXXXX-X (Universal Analytics)
ℹ️ Toggle this ON after entering your tracking ID to start tracking
📋 Setup Steps:
  1. Get tracking ID from Google Analytics
  2. Paste it above and toggle ON
  3. Click "💾 Save All Settings"
  4. Click "View Integration Code" button
  5. Copy and add code to your index.html

🔍 Google Search Console

Verify your website ownership to appear in Google search results. Get verification code from Google Search Console.

You can paste either:
• Just the code: abc123def456...
• Or the full meta tag: <meta name="google-site-verification" content="...">
📋 Setup Steps:
  1. Go to Google Search Console
  2. Add your property (http://tamilpetti.com)
  3. Choose "HTML tag" verification method
  4. Copy the meta tag or just the content code
  5. Paste it above and save settings
  6. Add generated code to index.html
  7. Go back to GSC and click "Verify"

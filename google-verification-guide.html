  1. Click on "HTML file" tab

  3. Upload this file to your website's root directory
  4. Test it works by visiting:
    http://tamilpetti.com/google1234567890abcdef.html
    (Replace with your actual filename)
  5. Go back to Google Search Console and click "Verify"

  4. Push changes to your website

üÜò Troubleshooting

Problem: Verification Failed

Solution:

  • Make sure the verification file is in the root directory (not in a subfolder)
  • Test the URL in your browser first
  • Wait 5 minutes and try verifying again
  • Try the HTML tag method instead

Problem: Sitemap Not Found

Solution:

  • Make sure sitemap.xml is uploaded to root directory
  • Test it by visiting: http://tamilpetti.com/sitemap.xml
  • Check that it's not blocked in robots.txt
  • Wait 24 hours and check again

Problem: Site Not Showing in Google After 2 Weeks

Solution:

  • Check "Coverage" report in Search Console for errors
  • Make sure robots.txt allows indexing
  • Request indexing again for main pages
  • Check if there are any manual penalties
